# LINUX : I/O


<br>

* I/O = Input/Output

* Disk IO

* Input = écriture

* Output = lecture

* data transfert

* point important de performance :
		* base de données
		* traitement de tâches généralement

Représentation :

	USER > Kernel > Mémoire <-> Disks

	Mémoire <-> Disks
		* Buffer = écriture
		* Cache  = lecture
	

* stockage = HDD / SSD / NAS (NET)

* IOPS => lecture/écriture par seconde
